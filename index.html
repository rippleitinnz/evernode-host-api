01 — Quick Start

Start querying in seconds

No authentication required. All endpoints return JSON. CORS enabled for browser requests.

# Get active hosts with high reputation and available slots curl "https://api.onledger.net/hosts?active=true&minRep=200&minSlots=1&limit=10" # Filter by country and sort by lease price curl "https://api.onledger.net/hosts?country=DE&minXah=5&minEvr=5&sortBy=leaseDrops&sortDir=asc" # Get a single host's full data curl "https://api.onledger.net/hosts/rnpkkEEMDSYAg1G6eHWF66kKjrKoAqMbtV" # Network statistics curl "https://api.onledger.net/stats"
02 — Endpoints

API reference

Expand ▼
GET/hostsFiltered host list

Returns a paginated list of hosts matching the specified filters. All parameters optional. Defaults: minXah=5, minEvr=5.

ParameterTypeDefaultDescription
activeboolean—Filter by active status
minSlots / maxSlotsinteger—Available instance slot range
minRep / maxRepinteger—Reputation score range (0–255)
includeUnscoredbooleanfalseInclude hosts with rep=0
minXahnumber5Minimum XAH balance
minEvrnumber5Minimum EVR balance
minLease / maxLeaseinteger—Lease price range in drops
countrystring—2-letter ISO country code (DE, FR, JP...)
versionstring—Sashimono version (1.0.0, 0.12.1...)
minRam / minDiskinteger—Minimum RAM or disk in MB
sortBystringhostReputationhostReputation, availableInstances, leaseDrops, xahBalance, evrBalance, ramMb, diskMb, lastHeartbeatIndex
sortDirstringdescasc or desc
limit / offsetinteger100 / 0Pagination
GET/hosts/:addressSingle host full data

Returns complete data for a single host by XRPL address. All fields including hardware specs, reputation, balances and registration details.

GET/hosts/randomRandom host sample

Returns a randomly selected set of hosts matching the given filters. Useful for load distribution across equivalent hosts without client-side shuffling.

ParameterTypeDefaultDescription
countinteger10Number of random hosts to return (max 200)
minRepinteger—Minimum reputation score
minSlotsinteger—Minimum available slots
countrystring—Filter by country code
activebooleantrueFilter by active status
GET/countriesHost counts by country

Returns all countries with registered hosts, including total host count, active host count and available slots per country. Useful for building country filter dropdowns or geographic dashboards.

GET/versionsSashimono version distribution

Returns all active Sashimono versions with host counts. Useful for filtering to a specific version or understanding network upgrade adoption.

GET/statsNetwork summary statistics

Network-wide statistics: total and active host counts, available slots, average reputation and balances, top countries by host count, and version distribution.

GET/healthAPI health check

Returns API status, host count, last full scan timestamp and whether a scan is currently in progress.

Query examples

Common queries shown in curl, browser fetch, and Node.js. All examples target https://api.onledger.net.

Find active hosts with available slots
The most common query — active hosts that have at least one free instance slot, sorted by reputation.
curl "https://api.onledger.net/hosts?active=true&minSlots=1&limit=20"
const res = await fetch('https://api.onledger.net/hosts?active=true&minSlots=1&limit=20'); const { hosts, total } = await res.json(); console.log(`${total} matching hosts`); hosts.forEach(h => console.log(h.address, h.availableInstances));
import fetch from 'node-fetch'; const res = await fetch('https://api.onledger.net/hosts?active=true&minSlots=1&limit=20'); const { hosts } = await res.json(); const addresses = hosts.map(h => h.address); console.log(addresses);
High-reputation hosts only
Filter to hosts with a reputation score above 240 — the upper tier of the 0–255 range.
curl "https://api.onledger.net/hosts?active=true&minRep=240&limit=50"
const res = await fetch('https://api.onledger.net/hosts?active=true&minRep=240&limit=50'); const { hosts } = await res.json(); // shuffle for random selection hosts.sort(() => Math.random() - 0.5); const picked = hosts.slice(0, 10); console.log(picked.map(h => h.address));
const res = await fetch('https://api.onledger.net/hosts?active=true&minRep=240&limit=500'); const { hosts } = await res.json(); // Fisher-Yates shuffle for true random sample for (let i = hosts.length - 1; i > 0; i--) { const j = Math.floor(Math.random() * (i + 1)); [hosts[i], hosts[j]] = [hosts[j], hosts[i]]; } const random200 = hosts.slice(0, 200); console.log(random200.map(h => h.address));
Filter by country
Find active hosts in a specific country using the 2-letter ISO country code.
# New Zealand hosts, sorted by cheapest lease first curl "https://api.onledger.net/hosts?active=true&country=NZ&sortBy=leaseDrops&sortDir=asc"
const params = new URLSearchParams({ active: 'true', country: 'NZ', sortBy: 'leaseDrops', sortDir: 'asc' }); const res = await fetch(`https://api.onledger.net/hosts?${params}`); const { hosts } = await res.json(); console.log(hosts);
const url = new URL('https://api.onledger.net/hosts'); url.searchParams.set('active', 'true'); url.searchParams.set('country', 'NZ'); url.searchParams.set('sortBy', 'leaseDrops'); url.searchParams.set('sortDir', 'asc'); const { hosts } = await fetch(url).then(r => r.json()); console.log(`Found ${hosts.length} NZ hosts`);
Hosts with sufficient resources
Find hosts meeting minimum hardware requirements — useful before deploying a contract.
# At least 4GB RAM, 20GB disk, 1 free slot curl "https://api.onledger.net/hosts?active=true&minRam=4096&minDisk=20000&minSlots=1&sortBy=ramMb&sortDir=desc"
const res = await fetch( 'https://api.onledger.net/hosts?active=true&minRam=4096&minDisk=20000&minSlots=1&sortBy=ramMb&sortDir=desc' ); const { hosts, total } = await res.json(); console.log(`${total} hosts meet requirements`); console.log(hosts.map(h => ({ address: h.address, ram: h.ramMb, disk: h.diskMb })));
const params = { active: 'true', minRam: 4096, minDisk: 20000, minSlots: 1, sortBy: 'ramMb', sortDir: 'desc', limit: 50 }; const qs = new URLSearchParams(params).toString(); const { hosts } = await fetch(`https://api.onledger.net/hosts?${qs}`).then(r => r.json()); console.log(hosts);
Paginate through all hosts
Use limit and offset to page through the full host list in batches.
# Page 1 (first 100) curl "https://api.onledger.net/hosts?active=true&limit=100&offset=0" # Page 2 (next 100) curl "https://api.onledger.net/hosts?active=true&limit=100&offset=100"
async function getAllHosts() { const pageSize = 100; let offset = 0, all = []; while (true) { const res = await fetch( `https://api.onledger.net/hosts?active=true&limit=${pageSize}&offset=${offset}` ); const { hosts } = await res.json(); if (!hosts.length) break; all = all.concat(hosts); offset += pageSize; } return all; }
async function getAllHosts() { const pageSize = 100; let offset = 0, all = [], done = false; while (!done) { const { hosts, total } = await fetch( `https://api.onledger.net/hosts?active=true&limit=${pageSize}&offset=${offset}` ).then(r => r.json()); all = all.concat(hosts); offset += pageSize; done = all.length >= total; } console.log(`Fetched ${all.length} hosts`); return all; } getAllHosts();

const address = 'rnpkkEEMDSYAg1G6eHWF66kKjrKoAqMbtV'; const host = await fetch(`https://api.onledger.net/hosts/${address}`) .then(r => r.json()); console.log(host);

05 — Data Fields

Host object

Every host object returned by the API contains the following fields.

{ "address": "rXXX..." // XRPL address "active": 1 // 1=active, 0=inactive "domain": "host.example.com" // host domain "countryCode": "NZ" // ISO 3166-1 alpha-2 "maxInstances": 3 // total slots configured "activeInstances": 1 // currently in use "availableInstances": 2 // free slots "leaseAmount": "0.001" // EVR per moment (1hr) "leaseDrops": 1000 // drops (1 EVR = 1,000,000) "hostReputation": 252 // 0–255, tested by reputation contract "version": "1.0.0" // sashimono version "cpuModelName": "AMD EPYC..." // CPU model "cpuCount": 4 // allocated CPU cores "cpuMHz": 3200 // clock speed MHz "ramMb": 8192 // RAM in MB "diskMb": 32000 // disk in MB "xahBalance": 34.5 // XAH balance "evrBalance": 12.3 // EVR balance "accumulatedReward": "1.23" // lifetime EVR rewards earned "lastHeartbeatIndex": 1775699062 // last heartbeat unix timestamp "registrationTimestamp": 1705310292 // when host registered "lastUpdated": 1775701701966 // last DB update (ms) }
